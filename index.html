"First Landing Page" Project

This was the first project assigned by the Odin Project curriculum. It allowed me to follow on the basics of HTML and CSS that I learned. The page was quite simple, although the use of flexbox was very interesting.

"Rock-Paper-Scissors" Project

At this point I was already getting more advanced. I basically recreated a Rock-Paper-Scissors game by using HTML, CSS and Javascript. The logic was definitely now simple, and I needed a bit of time to get it sorted. Nevertheless, the overall result was fairly creative, and without so many bugs...

"ETCH-A-SKETCH" Project

OK, so I remember this one. I needed a breakthrough moment due to not being able to properly resize the drawing grid based on user input. I remember wanting to ask the Discord community for help, although while looking for the 18th time at the code, I finally realized what I had to do. It was definitely a very good feeling to have.

"Calculator" Project

Ah, yes, the good old calculator. This was a bit annoying, I remember almost restarting it completely after about 6 hours of work. I thought the Javascript logic I applied was not good, although in the end I didn't need to do such a drastic refactoring.
